# 1. Project setup

1. **Create a virtual environment:**
    ```bash
    python3 -m venv .venv
    ```

2. **Activate the environment:**
    ```bash
    source .venv/bin/activate
    ```

3. **Install the requirements:**
    ```bash
    pip install -r requirements.txt
    ```

4. **Deactivate the environment:**
    ```bash
    deactivate
    ```
